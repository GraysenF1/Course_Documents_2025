This session is an OpenRefine workshop.

Download OpenRefine: http://openrefine.org/download.html
If you have a Windows operating system, and do not have Java installed, download the version labeled "Windows kit with embedded Java"
Install OpenRefine.

OpenRefine requires one of these browsers: Google Chrome, Chromium, Opera, or Microsoft Edge
It will not work with Internet Explorer and is not supported in Firefox.

If you need a new browser:
Download and install a browser from the approved list if you don't already have one.
You do not need to make the browser your default browser.
Use the new browser to point at http://127.0.0.1:3333/ or http://localhost:3333 to launch OpenRefine.
